**Mutations (3):**

4. `create` — mutation with args `{ clientId: v.id("clients"), programId: v.id("programs"), status: statusValidator, enrollmentDate: v.number(), exitDate: v.optional(v.number()), exitReason: v.optional(v.string()), notes: v.optional(v.string()) }`.
   - Call `requireRole(ctx, "admin", "manager", "staff")` — lawyers/psychologists do NOT create enrollments.
   - Insert into "enrollments" with all args + `createdBy: currentUser._id` + `updatedAt: Date.now()`.
   - CRITICAL: `updatedAt` is REQUIRED by schema — omitting it causes a runtime validation error.
   - Call `ctx.runMutation(internal.auditLog.log, { userId: currentUser._id, action: "create_enrollment", entityType: "enrollments", entityId: enrollmentId, details: ... })`.
   - Return `enrollmentId`.

5. `update` — mutation with args `{ enrollmentId: v.id("enrollments"), status: v.optional(statusValidator), exitDate: v.optional(v.number()), exitReason: v.optional(v.string()), notes: v.optional(v.string()) }`.
   - Call `requireRole(ctx, "admin", "manager", "staff")`.
   - Get existing record, throw "Enrollment not found" if missing.
   - Build sparse `updates` object (only set fields that are `!== undefined`), always include `updatedAt: Date.now()`.
   - Call `ctx.db.patch(args.enrollmentId, updates)`.
   - Audit log with action "update_enrollment".

6. `remove` — mutation with args `{ enrollmentId: v.id("enrollments") }`.
   - Call `requireRole(ctx, "admin")` — only admins delete enrollments.
   - Get existing, throw if missing.
   - Check for linked sessions: query sessions `withIndex("by_enrollmentId", ...)`. If any exist, throw "Cannot delete enrollment with linked sessions. Delete sessions first."
   - Call `ctx.db.delete(args.enrollmentId)`.
   - Audit log with action "delete_enrollment".

**Internal mutation (1):**

7. `importBatch` — internalMutation with args `{ enrollments: v.array(v.object({ clientId: v.id("clients"), programId: v.id("programs"), status: statusValidator, enrollmentDate: v.number(), exitDate: v.optional(v.number()), exitReason: v.optional(v.string()), notes: v.optional(v.string()), createdBy: v.id("users") })) }`.

   - Return `{ created, skipped }`.

/**
 * List sessions for a specific enrollment.
 */
export const listByEnrollment = query({
  args: { enrollmentId: v.id("enrollments") },
  handler: async (ctx, args) => {
    return await ctx.db
      .query("sessions")
      .withIndex("by_enrollmentId", (q) => q.eq("enrollmentId", args.enrollmentId))
      .collect();
  },
});
