- Check that Convex _generated/ files exist (they should after Step 1)

**Step 3: Deploy to VPS via rsync**

Run the exact commands from CLAUDE.md:

```bash
# Set password for sshpass
export SSHPASS="mnkC93'ksa5'HRikz2W@"

# Rsync standalone build (preserves .env.local and ecosystem.config.cjs on server)
sshpass -e rsync -avz --delete \
  ".next/standalone/" root@187.77.19.63:/var/www/webapp/ \
  --exclude='.env.local' --exclude='ecosystem.config.cjs' \
  -e "ssh -o StrictHostKeyChecking=no"

# Rsync static assets
sshpass -e rsync -avz ".next/static/" root@187.77.19.63:/var/www/webapp/.next/static/ \
  -e "ssh -o StrictHostKeyChecking=no"

# Rsync public dir
sshpass -e rsync -avz "public/" root@187.77.19.63:/var/www/webapp/public/ \
  -e "ssh -o StrictHostKeyChecking=no"
```

CRITICAL: The --exclude flags for .env.local and ecosystem.config.cjs prevent overwriting the server's environment variables and PM2 config. Do NOT remove these excludes.
