  <files>convex/analytics.ts, src/hooks/useAnalytics.ts</files>
  <action>
1. Create `convex/analytics.ts` with three public queries. These are simple aggregate queries with no auth requirement (dashboard is already behind auth layout).

   **getActiveClientCount:**
   - Query the `clients` table, filter by `status === "active"`, return the count.
   - Return shape: `{ count: number }`
   - Pattern: `const clients = await ctx.db.query("clients").collect(); const active = clients.filter(c => c.status === "active"); return { count: active.length };`

   **getSessionVolume:**
   - Query the `sessions` table, filter sessions where `sessionDate >= Date.now() - 30 * 24 * 60 * 60 * 1000`.
   - Return shape: `{ count: number, periodLabel: "Last 30 days" }`
   - Pattern: Similar to existing `getActiveSessions` in sessions.ts but returns count instead of array.

   **getIntakeTrend:**
   - Compute start of current month and start of previous month using `new Date()`.
   - Query `legalIntakeForms` table, filter by `createdAt >= startOfCurrentMonth` for this month count, and `createdAt >= startOfPrevMonth && createdAt < startOfCurrentMonth` for last month count.
   - Do the same for `coparentIntakeForms`.
   - Sum both form types for each period.
   - Return shape: `{ thisMonth: number, lastMonth: number, changePercent: number, positive: boolean }`
   - `changePercent`: If lastMonth is 0 and thisMonth > 0, return 100. If both 0, return 0. Otherwise `Math.round(((thisMonth - lastMonth) / lastMonth) * 100)`.
   - `positive`: `thisMonth >= lastMonth` (more intakes = positive for a nonprofit serving clients).

   All three queries use `import { query } from "./_generated/server"` and `import { v } from "convex/values"`. Args should be empty objects `args: {}`.

2. Create `src/hooks/useAnalytics.ts`:
   ```typescript
   import { useQuery } from "convex/react";
   import { api } from "../../convex/_generated/api";

   export function useActiveClientCount() {
     return useQuery(api.analytics.getActiveClientCount);
   }

   export function useSessionVolume() {
     return useQuery(api.analytics.getSessionVolume);
   }

   export function useIntakeTrend() {
     return useQuery(api.analytics.getIntakeTrend);
   }
   ```

  </verify>
