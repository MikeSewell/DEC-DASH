**1. `getSessionTrends` query:**
- Compute the past 12 months as an array of `{ label: string, start: number, end: number }` where `label` is "MMM YYYY" format (e.g. "Mar 2026"), `start` is epoch ms for the 1st of that month, `end` is epoch ms for the 1st of the next month.
- Build months going backward from the current month: `new Date()` → get year/month → loop 12 times decrementing month (handle year wrap).
- Collect all sessions from the table: `ctx.db.query("sessions").collect()`.
- For each month bucket, count sessions where `sessionDate >= start && sessionDate < end`.
- Return `{ months: Array<{ label: string; count: number }> }` — array is chronological (oldest first).
- Example return: `{ months: [{ label: "Apr 2025", count: 3 }, { label: "May 2025", count: 5 }, ...] }`

**2. `getGoalStats` query:**
- Collect all client goals: `ctx.db.query("clientGoals").collect()`.
- Count by status: `in_progress`, `completed`, `not_started`.
- Compute `completionRate`: `Math.round((completed / total) * 100)` — if total is 0, set to 0.
- Return `{ inProgress: number, completed: number, notStarted: number, total: number, completionRate: number }`.

**3. `getIntakeVolume` query:**
- Same 12-month bucket array as `getSessionTrends` (factor out the month-generation into a helper function at the top of the file, something like `function getLast12Months()` — NOT exported, just a local helper).
- Use `by_createdAt` index range queries for efficient filtering (same pattern as the existing `getIntakeTrend` query):
  - For each month, query `legalIntakeForms` with `.withIndex("by_createdAt", q => q.gte("createdAt", start).lt("createdAt", end))`.
  - Same for `coparentIntakeForms`.
- Return `{ months: Array<{ label: string; legal: number; coparent: number }> }` — chronological order.

**Helper function `getLast12Months`:**
```typescript
function getLast12Months(): Array<{ label: string; start: number; end: number }> {
  const now = new Date();
  const months: Array<{ label: string; start: number; end: number }> = [];
  for (let i = 11; i >= 0; i--) {
    const d = new Date(now.getFullYear(), now.getMonth() - i, 1);
    const next = new Date(d.getFullYear(), d.getMonth() + 1, 1);
    const label = d.toLocaleDateString("en-US", { month: "short", year: "numeric" });
    months.push({ label, start: d.getTime(), end: next.getTime() });
  }
  return months;
}
```
Place this helper AFTER the existing imports and BEFORE the first `export const`.

**For `getSessionTrends`**, use `getLast12Months()` and iterate sessions. Sessions use `sessionDate` (not `createdAt`), and no index is available on `sessionDate`, so collect all and filter in-memory (same pattern as `getSessionVolume`).

**For `getIntakeVolume`**, use `getLast12Months()` but leverage the `by_createdAt` index for efficient per-month queries (do NOT collect all then filter).
