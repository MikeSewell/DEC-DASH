**Section 1: Session Volume Trends (ACT-01)**
- Full-width card: `rounded-2xl border border-border bg-surface p-5 shadow-[var(--warm-shadow-sm)]`
- Title: `<h4 className="text-sm font-semibold text-foreground mb-3">Session Volume (Past 12 Months)</h4>`
- Chart: `<Line>` component with `height: 280`.
- Data: labels from `sessionTrends.months.map(m => m.label)`, single dataset with `data: sessionTrends.months.map(m => m.count)`.
- Dataset styling: `borderColor: "#1B5E6B"` (primary), `backgroundColor: "rgba(27,94,107,0.1)"`, `fill: true`, `tension: 0.3` (smooth curve), `pointRadius: 4`, `pointBackgroundColor: "#1B5E6B"`, `borderWidth: 2`.
- Options: `responsive: true`, `maintainAspectRatio: false`, plugins: `{ legend: { display: false }, tooltip: CHART_TOOLTIP }`, scales x: `{ grid: { display: false }, ticks: { font: { family: "'Nunito', sans-serif", size: 11 } } }`, scales y: `{ beginAtZero: true, grid: { color: "rgba(45,106,79,0.06)" }, ticks: { font: { family: "'Nunito', sans-serif", size: 11 }, precision: 0 } }`.
- If all month counts are 0, show an empty state message: "No session data recorded yet." inside the card (below the title, no chart).

**Section 2: Goal Status Breakdown (ACT-02)**
- Grid row with 4 stat cards (similar to DemographicsTab summary row pattern) using `grid grid-cols-2 sm:grid-cols-4 gap-4`.
- Card 1: Completed goals — value = `goalStats.completed`, label = "Completed", text color `text-success`.
- Card 2: In Progress — value = `goalStats.inProgress`, label = "In Progress", text color `text-primary`.
- Card 3: Not Started — value = `goalStats.notStarted`, label = "Not Started", text color `text-muted`.
- Card 4: Completion Rate — value = `goalStats.completionRate + "%"`, label = "Completion Rate", text color `text-accent`.
- Each card: `rounded-2xl border border-border bg-surface p-4 text-center shadow-[var(--warm-shadow-sm)] hover-lift`.
- Value: `<p className="text-xl font-bold {color}">{value}</p>`.
- Label: `<p className="text-xs text-muted">{label}</p>`.
- If `goalStats.total === 0`, show a single full-width card with "No client goals recorded yet." message instead of the 4-card grid.

**Section 3: Intake Volume by Month (ACT-03)**
- Full-width card matching session chart pattern.
- Title: `<h4 className="text-sm font-semibold text-foreground mb-3">Intake Volume (Past 12 Months)</h4>`
- Chart: `<Bar>` component (vertical, NOT horizontal) with `height: 280`.
- Data: labels from `intakeVolume.months.map(m => m.label)`.
- Two datasets:
  - Legal: `{ label: "Legal", data: intakeVolume.months.map(m => m.legal), backgroundColor: "#1B5E6B", borderRadius: 6 }`
  - Co-Parent: `{ label: "Co-Parent", data: intakeVolume.months.map(m => m.coparent), backgroundColor: "#6BBF59", borderRadius: 6 }`
- Options: `responsive: true`, `maintainAspectRatio: false`, plugins: `{ legend: { position: "top", labels: { usePointStyle: true, pointStyle: "circle", padding: 12, font: { size: 11, family: "'Nunito', sans-serif" } } }, tooltip: CHART_TOOLTIP }`, scales x: `{ stacked: false, grid: { display: false }, ticks: { font: { family: "'Nunito', sans-serif", size: 11 } } }`, scales y: `{ stacked: false, beginAtZero: true, grid: { color: "rgba(45,106,79,0.06)" }, ticks: { font: { family: "'Nunito', sans-serif", size: 11 }, precision: 0 } }`.
- Bars should be GROUPED (side by side), not stacked. `stacked: false` on both axes ensures this.
- If all legal and coparent values are 0, show empty state: "No intake data recorded yet."

**Overall layout:** Wrap all three sections in `<div className="space-y-6">`. The order is:
1. Session Volume chart (full width)
2. Goal Status cards (grid row)
3. Intake Volume chart (full width)
