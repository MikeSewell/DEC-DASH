<task id="28-02-T3" title="Add urgency color coding to CalendarWidget event rows">
<description>
Enhance the `CalendarWidget` in `src/components/dashboard/CalendarWidget.tsx` to apply urgency-based background coloring to event rows based on how soon the event occurs, in addition to the existing type-based badge coloring.

**Add an urgency helper function:**

```typescript
function getEventUrgencyClasses(startAt: number, now: number): string {
  const diffMs = startAt - now;
  const diffDays = diffMs / 86400000;

  if (diffDays < 0) {
    // Past event / currently happening — no urgency tint (already has border-l for "happening")
    return "";
  }
  if (diffDays <= 1) {
    // Today or tomorrow — red tint (imminent)
    return "bg-red-500/[0.03] dark:bg-red-500/[0.06]";
  }
  if (diffDays <= 3) {
    // Within 3 days — amber tint (approaching)
    return "bg-amber-500/[0.03] dark:bg-amber-500/[0.06]";
  }
  // 4+ days away — green tint (comfortable), very subtle
  return "bg-emerald-500/[0.02] dark:bg-emerald-500/[0.04]";
}
```
